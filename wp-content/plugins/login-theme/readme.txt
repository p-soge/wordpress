== Description ==

Nice and clean login theme for wordpress default login page, easy to customize and select different themes.

### Login Theme by http://paratheme.com

<strong>Plugin Features</strong><br />

* Custom logo.
* Three different theme.
* Custom background color login area.
* Custom background image login page.
* Custom background color for submit button.

== Installation ==

1. Install as regular WordPress plugin.<br />
2. Go your plugin setting via WordPress Dashboard and find "<strong>Login Theme</strong>" activate it.<br />

After activate plugin you will see "Login Theme" menu at left side.
